## Project
1. Clone Project:
```
git clone https://github.com/furqonaugust17/PBL2D-Kel6-Project.git
```

2. Install Dependency
```
Composer Install
```

3. Create .ENV
```
cp .env.example .env
```

4. Generate Application Key
```
php artisan key:generate
```

5. Run Migrations
```
php artisan migrate --seed
```

6. Run Application
```
php artisan serve
```
